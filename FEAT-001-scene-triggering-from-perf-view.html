Feature Request #001: Scene Triggering from PERF View

Product: OXI ONE MKII
Firmware Version: 0.17.7
Date Created: November 15, 2025

Persona

Live Performer

Problem Statement

As a live performer, I need to trigger arranger scenes (clip launch of all sequencers) directly from PERF (Performance Grid) view. Currently, I have to switch to Arranger view to trigger scenes, which is disruptive during live performance. Switching between grid views with different layouts is disorienting, especially in Arranger view where it's hard to see where I am currently, even with visual feedback for the playing scene.

Proposed Solution

Overview

Configure columns 9-16 in PERF view to trigger arranger scenes (up to 32 scenes across 8 columns Ã— 4 rows), while keeping columns 1-8 reserved for mute/unmute controls.

Example showing currently playing scene: SC3 is highlighted with white background

Legend:

  • S1-S8 = Sequencer rows (each row corresponds to one of the 8 sequencers)
  • M = Mute/Unmute - Columns 1-8, mute/unmute individual sequencers
  • SC = Scene trigger - Triggers all 8 sequencers simultaneously (equivalent to SHIFT+column shortcut)
  • Empty cells = Not used for scene triggering in this layout

Scene Trigger States:

  1. Active (e.g., SC3) - Currently playing scene
  2. Inactive (e.g., SC1, SC2, SC4-6, SC9-13, etc.) - At least one pattern assigned
  3. Not Used (e.g., SC7-8, SC14-16, SC20, SC24, SC28-32) - No patterns assigned

Additional Visual Feedback:

  • Queued/next scene can be indicated visually

Benefits

  1. Improved Performance Flow: Stay in PERF view without switching to Arranger view for scene triggering
  2. Better Orientation: Consistent grid layout reduces cognitive load during live performance
  3. Faster Scene Access: Direct scene triggering from performance view
  4. Flexibility: Multiple layout options accommodate different performance styles
